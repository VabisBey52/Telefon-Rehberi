# 📞 Telefon Rehberi - İlk C# Projem

Bu uygulama, yazılım geliştirme serüvenimdeki ilk **C# Windows Forms** projemdir. Temel kullanıcı etkileşimlerini öğrenmek ve verileri form elemanları üzerinden yönetmek amacıyla geliştirilmiştir.

---

### 🚀 Özellikler

- **Veri Girişi:** TextBox'lar aracılığıyla isim, soyisim ve telefon numarası bilgilerini alma.
- **Kullanıcı Dostu Arayüz:** Sade ve anlaşılır form tasarımı.
- **Dinamik Yönetim:** Girilen verilerin listelenmesi ve yönetilmesi.
